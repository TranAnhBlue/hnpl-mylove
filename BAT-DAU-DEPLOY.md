# 🚀 BẮT ĐẦU DEPLOY

## ⚡ Cách Nhanh Nhất

### Chạy Script Tự Động:
```bash
DEPLOY-NOW.bat
```

Script sẽ:
- ✅ Kiểm tra Node.js, npm, Git
- ✅ Cài đặt Vercel CLI
- ✅ Cài đặt tất cả dependencies
- ✅ Build frontend
- ✅ Deploy lên onlylovehnpl.vercel.app
- ✅ Hướng dẫn deploy backend

---

## 📋 Hoặc Làm Thủ Công

### Bước 1: Cài Vercel CLI
```bash
npm install -g vercel
```

### Bước 2: Login Vercel
```bash
vercel login
```

### Bước 3: Deploy
```bash
deploy-onlylove.bat
```
