# 🎯 BẮT ĐẦU TỪ ĐÂY

## 🚀 Deploy lên onlylovehnpl.vercel.app

### ⚡ Cách Nhanh Nhất (5 phút)

#### Bước 1: Cài Vercel CLI
```bash
npm install -g vercel
```

#### Bước 2: Login Vercel
```bash
vercel login
```

#### Bước 3: Deploy!
```bash
# Windows
deploy-onlylove.bat

# Mac/Linux
chmod +x deploy-onlylove.sh
./deploy-onlylove.sh
```

**Xong!** Ứng dụng sẽ online tại: https://onlylovehnpl.vercel.app
